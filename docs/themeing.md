# Theme Customization

All components use CSS variables.

Override them globally.

```css
:root {
 --bg-color: #ffffff;

 --font-color: #000000;

 --border-color: #000000;

 --border-focus-color: #2563eb;

 --bg-disabled: #a3a3a3;

 --label-size: 16px;

 --label-weight: 600;

 --content-size: 14px;

 --content-weight: 400;

 --min-padding: 0.2rem;

 --border-radius: 0.2rem;

 --bg-active: #8a9ac7;

 --success-color: #2cca46;

 --danger-color: #e43d3d;
}
```

## Variables

| Variable             | Description          |
| -------------------- | -------------------- |
| --bg-color           | Component background |
| --font-color         | Text color           |
| --border-color       | Border color         |
| --border-focus-color | Focus border         |
| --bg-disabled        | Disabled state       |
| --label-size         | Label font size      |
| --label-weight       | Label weight         |
| --content-size       | Content font size    |
| --content-weight     | Content weight       |
| --min-padding        | Default spacing      |
| --border-radius      | Border radius        |
| --bg-active          | Active state         |
| --success-color      | Success color        |
| --danger-color       | Error color          |

# Security Policy

Thank you for taking the time to report security vulnerabilities and help improve this project.

## Supported Versions

The latest released version of this package is actively supported with security updates.

| Version        | Supported |
| -------------- | --------- |
| Latest         | ✅ Yes    |
| Older versions | ❌ No     |

Older versions are no longer maintained and may contain known issues. Please upgrade to the latest release.

## Reporting a Vulnerability

If you discover a security vulnerability, **please do not create a public GitHub issue**.

Instead, report it privately using one of the following methods:

### 📧 Email

**s.chandrasekhar.h@gmail.com**

### 💻 GitHub

Repository: https://github.com/CS7player/x-library

If GitHub **Private Vulnerability Reporting** is enabled, please use that feature instead of opening a public issue.

## What to Include

To help investigate the issue quickly, please include:

- Package name and version
- Node.js version
- Operating system
- Steps to reproduce the issue
- Proof of concept (if available)
- Potential impact

## Response Timeline

- Initial acknowledgment: **within 48 hours**
- Status updates: As progress is made
- Security fixes will be released as soon as possible after verification.

## Disclosure Policy

Please allow adequate time for the vulnerability to be investigated and fixed before making any public disclosure.

Thank you for helping keep **x-library** secure! ⭐
